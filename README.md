# Student Result System

## 📌 About the Project

This is a simple Python project that calculates a student's total marks, average, grade, and final result (Pass/Fail).

## ✨ Features

* Accepts student name and marks for 3 subjects
* Calculates total marks
* Calculates average marks
* Assigns grade based on performance
* Displays final result (PASS / FAIL)

## 🛠️ Technologies Used

* Python

This project helped me understand:

* Input and output handling
* Arithmetic operations
* Conditional statements
* Basic problem-solving in Python
